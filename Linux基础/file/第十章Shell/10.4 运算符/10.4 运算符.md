# 10.4 运算符

### 4.1 数学计算式语法

1. 基本语法

“$((运算式))” 或 “$[运算式]”

2. 参考实例

计算（2+3）* 4 的值   

```shell
S=$[(2+3)*4]
echo $S
```

### 4.2 获取命令和函数的返回值

1. 基本语法

* 使用两个单引号包裹的命令可以用变量取到返回值
* 使用$()包裹的命令可以用变量取到返回值

2. 参考实例

* 方式一

```shell
a=`date`
```

变量a中存放着命令date的返回值

方式二

```shell
a=$(date)
```

变量a中存放着命令date的返回值

方式三

```shell
function getStr() {
    echo "hello"
}

a=$(getStr)
```

变量a中存放着函数getStr的返回值
