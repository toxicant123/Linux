# 10.7 read 读取控制台输入

1. 基本语法

read (选项) (变量)

选项：
* -p：指定读取值时的提示符；
* -t：指定读取值时等待的时间（秒）如果-t 不加表示一直等待

变量：指定读取值的变量名

2. 案例实操

提示 7 秒内，读取控制台输入的名称

```shell
touch read.sh
vim read.sh
##!/bin/bash
#read -t 7 -p "Enter your name in 7 seconds :" NN
#echo $NN
./read.sh
#Enter your name in 7 seconds : atguigu
#atguigu
```
